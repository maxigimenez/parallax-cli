# @parallax/marketing

Detached marketing site package for Vercel deployment.

## Scope

- Standalone Next.js + Tailwind app.
- No dependency on `@parallax/cli` or `@parallax/ui`.
- Reads documentation content from repository `/docs/*.md`.

## Local development

```bash
pnpm --filter @parallax/marketing dev
```

## Build

```bash
pnpm --filter @parallax/marketing build
```
